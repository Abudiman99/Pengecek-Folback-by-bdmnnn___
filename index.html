Cara Mendapatkan Data dari Instagram

Ada dua cara untuk mendapatkan daftar pengikut dan yang Anda ikuti:

Cara 1: Salin Manual (Cepat)

Ini cara tercepat jika daftarnya tidak terlalu panjang. Buka aplikasi Instagram di HP Anda, pergi ke profil, lalu ketuk **"Mengikuti"** atau **"Pengikut"**. Gulir ke bawah hingga semua nama termuat, lalu salin dan tempel datanya langsung ke kotak teks di atas.

Cara 2: Minta Salinan Data Resmi (Terbaik)

Ini cara paling akurat karena Instagram akan mengirimkan seluruh data Anda dalam satu file ZIP.

  • Buka **Pengaturan dan privasi** di aplikasi Instagram.
  • Pilih **Pusat Akun** → **Informasi dan izin Anda** → **Unduh informasi Anda**.
  • Ketuk **Minta unduhan** → pilih akun Anda → **Beberapa informasi** → centang **Pengikut dan Mengikuti** → **Format HTML**.
  • Tunggu email dari Instagram dengan tautan unduhan. Setelah diunduh, ekstrak file ZIP dan cari file **followers.html** dan **following.html**.
  • Buka kedua file tersebut di browser dan salin isinya ke kotak teks di atas.
